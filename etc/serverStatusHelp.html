Enstore System Status Help

Overview

Enstore Server Status shows the most recent known state of the Enstore system. Each server, mover, library manager and media changer is represented. Since there may be many entries in the status table, a shortcut table is listed at the top. This page will automatically refresh, based on the value listed in the inquisitor section of the configuration file. It is created by the inquisitor, and includes the time/date when it was last updated in its upper right hand corner.

Shortcuts

There are four types of entries in the shortcut table -
Library Managers
Each library manager will have a separate entry in the shortcut table.
Movers
This link points to the section which lists all of the movers (in alphabetical order)
Unmonitored Servers
This link points to the section which lists all of the servers (in alphabetical order) which are listed in the configuration file but which the inquisitor is not monitoring. Each individual server entry in the configuration file, will have inq_ignore in it to indicate that this server will not be monitored by the inquisitor.
Full File List
This link points to the page that contains a list of all of the user files that are currently being transferred by enstore.

Status Table

The status table shows the last known status of the servers, library managers, movers and media changers. In the following discussion the term element will be used to refer to any of these. Each element displays a State. This is the first row of information associated with each element. The library managers display this State and additional information. The color of the background for the name of an element (first column), represents a visual cue to the status of the element.

Everything is OK
A mover is alive but in one of the following states
  • OFFLINE
  • DRAINING
  • ERROR
A library manager is alive but in one of the following states
  • BROKEN
The element does not appear to be alive
The element is not being monitored
An error occurred while getting the elements status

Library Manager Additional Information

The additional Library Manager information includes the following -
StateInternal state of the library manager (added to the above mentioned state
Ongoing TransfersNumber of transfers currently assigned to Movers
Pending TransfersNumber of transfers not yet assigned to a Mover
Queue ElementsA list of the jobs (active and pending) of which the library manager is aware.

Only the number of queue elements as specified in the configuration file will be displayed on this page. If the total number of queue elements is larger than this value, the rest of the elements will be displayed on a separate page and a link to this page will be included at the end of the queue, Extra Queue Rows (n), where n is the number of elements on the additional page).
