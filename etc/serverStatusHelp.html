Enstore System Status Help

Overview

Enstore System Status shows the most recent known state of the Enstore system. Each server, mover, library manager and media changer is represented. Since there may be many entries in the status table, a shortcut table is listed at the top. This page will automatically refresh, based on the value listed in the inquisitor section of the configuration file. It is created by the inquisitor, and includes the time/date when it was last updated in its upper right hand corner.

Shortcuts

There are four types of entries in the shortcut table -
Servers
This link points to the section which contains the following servers -
  • alarm_server
  • config_server
  • file_clerk
  • inquisitor
  • log_server
  • volume_clerk
library managers
Each library manager will have a separate entry in the shortcut table.
Media Changers
This link points to the section which lists all of the media_changers (in alphabetical order)
Movers
This link points to the section which lists all of the movers (in alphabetical order)

Status Table

The status table shows the last known status of the servers, library managers, movers and media changers. In the following discussion the term element will be used to refer to any of these. Each element displays a Status. This is the first row of information associated with each element. The library managers and movers display this Status and additional information. The color of the background for the name of an element (first column), represents a visual cue to the status of the element.

Everything is OK
A mover is alive but in one of the following states
  • OFFLINE
  • DRAINING
  • ERROR
  • The element does not appear to be alive
  • The element is not being monitored
An error occurred while getting the elements status

Library Manager Additional Information

The additional Library Manager information includes the following -
StateInternal state of the library manager
Suspect VolumeList of Suspect Volumes and the Mover which reported them as suspect
Ongoing TransfersNumber of transfers currently assigned to Movers
Pending TransfersNumber of transfers not yet assigned to a Mover
Work At Movers QueueDetails on the work assigned to a Mover, including a link to that Mover
Pending QueueDetails on the work not yet assigned to a Mover

Only a configurable number of queue elements will be displayed on this page. If the total number of queue elements is larger than this value, the rest of the elements will be displayed on a separate page and a link to this page will be included at the end of the queue with which it is associated (Extra Queue Rows (n), n is the number of elements on the additional page).

Mover Additional Information

The additional Mover information includes the following -
Completed TransfersNumber of transfers this Mover has completed since the last restart
Failed TransfersNumber of failed transfers this Mover has encountered since the last restart
Current StateOne of the legal Mover states
Current/Last Read (bytes)How many bytes are/have been read in the current/last transfer
VolumeThe volume that is being read from or written to
Current/Last Write (bytes)How many bytes are/have been written in the current/last transfer
Location CookieThe location of the file on the tape
Source FileThe name of the file that is being read
Destination FileThe name of the file that is being written
