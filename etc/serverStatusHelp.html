Enstore System Status Help

Overview

Enstore System Status shows the most recent known state of the Enstore system. Each server, mover, library manager and media changer is represented. Since there may be many entries in the status table, a shortcut table is listed at the top. This page will automatically refresh, based on the value listed in the inquisitor section of the configuration file. It is created by the inquisitor, and include the time/date when it was last updated in the upper right hand corner.

Shortcuts

There are four types of entries in the shortcut table -
Servers
This link points to the section which contains the following servers -
  • alarm_server
  • config_server

  • inquisitor
  • log_server
  • volume_clerk
library managers
Each library manager will have a separate entry in the shortcut table.
Media Changers

Movers
This link points to the section which lists all of the movers (in alphabetical order

Status Table

The status table shows the last known status of the servers, library managers, movers and media changers. In the following discussion the term element will be used to refer to any of these. Each element displays a Status. The library managers and movers display this Status and additional information. The color of the background for the name of an element, represents a visual cue to the status of the element.

Everything is OK
A mover is alive but in one of the following states
  • OFFLINE
  • DRAINING
  • ERROR
The element was unreachable (the communications timed out)
An error occurred while getting the elements status
