Remote Access to FNAL Mass Storage Data Sets

The ISD department is commissioning a FTP service for the Enstore Mass Storage system.

The service runs on node fndca.fnal.gov, port 24126. The service can be accessed by ordinary FTP Client software but
you need specify port explicitly to access service (see example).

In order to establish the service user must register with the Computing division, by contacting the ISA group via enstore-admin@fnal.gov
You must have valid FNAL account (UID).

Administration of the service is via the ISA group in the ISD Department. Administrative requests can be sent to helpdesk@fnal.gov
