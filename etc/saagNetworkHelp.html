Network Status-At-A-Glance Help

Overview

Network Status-At-A-Glance provides the visual status of the Enstore network that connects each of the nodes running a server or a mover. Once a problem has been identified it is expected that the more complete and detailed view of the network will be examined.

There are 2 levels of status represented on this page -

  • No problems indicated by a
  • Minor problem indicated by a
A Minor problem will indicate that the transfer rate between the base node and the target node is 0, the target node is unreachable, or perhaps the monitor_server is not running on the target node.

Enstore Network Status

Each node mentioned in the configuration file is listed here. A node may however, be explicitly marked in the configuration file to be excluded from this measurement. An encp-like connection is made between the Base Node and each of the other nodes. The Base Node is the node that initiates all of the network communications used to evaluate the state of the network. The data communications rate between the two nodes is measured. If the non-base node does not respond to the connection or if the data transfer rate between the two is 0.0, then the node is marked as . Otherwise the node is marked .
