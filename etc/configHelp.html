Enstore Configuration Help

Overview

This page shows the current Enstore configuration as specified in the Enstore configuration file. There are 2 parts to the information, a shortcut table at the top of the page and the detailed configuration information. The shortcut table contains a link to the information for each of the servers that are listed in the configuration file. The shortcut links are arranged in alphabetical order.

The individual configuration information is arranged alphabetically by server and then (internal to each server), alphabetically by element. No interpretation of the element values is done, so if the value is a python dictionary, then it is presented here as such.
