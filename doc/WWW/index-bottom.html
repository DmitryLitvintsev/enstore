DCache Systems
  DCache provides disk-based front-end data buffering for the Enstore mass storage system. Files transferred to the dCache are written to the tape after a certain time limit or data accumulation, transparently to the user. When reading files from the dCache, users transparently access a file replica already in cache due to past activity or a file replica restored to cache from tape.
