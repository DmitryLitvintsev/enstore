Enstore provides distributed access to and management of data stored on tape. It provides a generic interface so experimenters can efficiently use mass storage systems as easily as if they were native file systems.

Data Cache provides front end data buffering for the Enstore mass storage system. Files transferred to the dCache are written to the tape in transparent for the user way. When reading file from the dCache user may get disk replica of the file if it happily sits there in the disk buffer due to the previous transfer or read ahead.
