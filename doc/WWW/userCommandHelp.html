Enstore User Command Help

Overview

The Enstore User Command page gives web access to Enstores' user commands. These commands allow non-intrusive access to Enstore to perform necessary functions. There are 4 major command groups -

  • File Clerk commands
  • Library Manager Commands
  • Volume Clerk commands
  • PNFS commands
To execute a command, the command group must be selected, the apropriate options entered and then pressed. If a mistake is made, pressing will clear all selections.

Only one command from a command group may be executed at a time. Selecting a second command group will automatically deselect any other command group. Each command group has access to help () for its particular commands.

Command Options Required Parameters Definition
file bfid=bit_file_id   get information on the specified bit file id
  help   get help information
  list=volume   get a file list of the specified volume
  recursive   recursively restore all files
  restore=file_name   restore the specified file
  retries=number   number of retries for udp communication
  timeout=seconds   number of seconds to wait for udp response
library delete_work=work_id library_manager_name remove the specified work from the queue of work to do
  get_queue node library_manager_name return the list of encp requests by node and library manager
  help   get help information
  priority=number library_manager_name set the priority for the specifed work in the queue of work to do
  retries=number   number of retries for udp communication
  timeout=seconds   number of seconds to wait for udp response
volume add=volume_name library file_family media_type volume_byte_capacity remaining_capacity add the specified volume to Enstore
  delete=volume_name   delete the specified volume from Enstore
  help   get help information
  new_library=library_name volume_name associate the specified volume with the specified library manager
  no_access=   do not allow access to the specified volume
  read_only=   do not allow writes to the specified volume
  restore=   restore the specified volume within Enstore
  retries=   number of retries for udp communication
  timeout=   number of seconds to wait for udp response
  update=   update the status of the volume
  vol=   get information on the specified volume
pnfs enstore_state   list whether Enstore is still accepting transfers
  files volmap-tape list all files on the specified tape in volmap
  file family [value] set file family tag to value
  file family width [value] set library tag to value
  file family wrapper [value] set file family wrapper tag to value
  library [value] set library tag to value
  tags directory list the tags in the directory
  xref file list the cross-reference info about the file
