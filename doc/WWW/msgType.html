'MSG_TYPE' Error Messages

Here is a partial listing of many msg_types error messages that you may grep on or use to search for the string character sequence in a program. All messages that users pass to the ENSTORE program will have a 'MSG_TYPE' appended to the end of the message. The user can also pass their own 'MSG_TYPE' if so desired. This is to make it easier for them to find a log entry. 'Device' or 'server' are in small letters to show that it changes. In reality it will be upper case of the device or server that is being affected. I.e. ADD_LIST_SUC_AITLBM.
