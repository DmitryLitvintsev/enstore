encp Offsite Test


To verify that encp (or enstore commands) do not work from offsite:
1) Identify a test node on site to run encp on.  Preferably, this would be
   a node with encp/enstore already installed.
2) Perform a test encp to make sure encp works.
3) Modify the configuration file of the Enstore system that will be used
   for the test.  In the 'domains' section there is a list called 
   'invalid_domains'.  Add the test node ip to this list.
4) Tell the config server to use the newly modified configuration file.
5) Rerun the encp from step two.  It should hang for a little while.  
   Eventually it will fail with the error CONFIGDEAD.

6) Reset the configuration file back to what it was.
